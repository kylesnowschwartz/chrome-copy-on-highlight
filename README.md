# Copy on Select

Chrome extension that automatically copies highlighted text to clipboard - terminal-style.

## Install

1. Clone this repo
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select this directory

## Usage

Select text on any webpage - it's copied automatically.

Hold **Ctrl** (or **Cmd** on Mac) while selecting to bypass auto-copy.

## Limitations

- Doesn't work on `chrome://` pages (Chrome restriction)
- Requires HTTPS or localhost (secure context)
- Some sites may block clipboard access via CSP
